安装¶

依赖¶

ScrapydArt 依赖于以下库(与Scrapyd保持一致，因此你不需要为此困扰)，并且版本不能低于指定版本:

  • Python 2.7 or above
  • Twisted 8.0 or above
  • Scrapy 1.0 or above
  • six

安装ScrapydArt¶

如何安装ScrapydArt取决平台. 通常我们都会从PyPI下载安装:

pip install scrapydart
